# API Reference

This section is the C# API reference for PKDSIM, generated with DocFX directly
from the source in the `PKDSIM/UnityAssets` submodule.

Use the table of contents on the left to browse by namespace. Everything is
rooted under `project_pkdsim.Assets.*`; the most useful starting points are:

- **`project_pkdsim.Assets.USys`** &mdash; the `USys` orchestrator.
- **`project_pkdsim.Assets.USys.*Sys`** &mdash; the individual runtime subsystems
  (money, dialogue, saving, teleporting, anti-cheat, &hellip;). See the
  [Subsystems article](../articles/subsystems.md) for what each one does.
- **`project_pkdsim.Assets.Editor`** &mdash; editor-only tooling (see
  [Editor workspaces](../articles/editor-workspaces.md)).

> [!NOTE]
> The API pages are generated and are **not** committed to the repository. Run
> `make docs` (or `./build.ps1`) to produce them locally &mdash; see
> [Building the docs](../articles/building-docs.md).
